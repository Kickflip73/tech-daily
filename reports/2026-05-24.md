## 五、🔒 安全 — NGINX Rift 漏洞震动全球

### ⚠️ CVE-2026-42945："NGINX Rift" — 潜伏 18 年的严重漏洞

| 属性 | 详情 |
|------|------|
| **CVE 编号** | CVE-2026-42945 |
| **代号** | NGINX Rift |
| **CVSS 评分** | **9.2 (Critical)** |
| **漏洞类型** | 堆缓冲区溢出（Heap Buffer Overflow） |
| **影响模块** | ngx_http_rewrite_module（URL 重写模块） |
| **影响版本** | NGINX 0.6.27 ~ 1.30.0（几乎全部版本） |
| **全球受影响** | 约 **570 万台服务器**，中国超 **254 万网站** |
| **发现者** | depthfirst 安全研究团队 |

**漏洞详情：**
该漏洞存在于 NGINX 最核心的 URL 重写模块中，自 2008 年引入后潜伏长达 18 年。攻击者可通过构造恶意请求触发堆溢出，导致服务器崩溃或远程代码执行（RCE）。披露后即被发现在野利用。

**修复措施：** 立即升级至 **NGINX 1.30.1+** 修复版本。

### Microsoft 5 月 Patch Tuesday

- 修复 **120 个漏洞**，含 **29 个远程代码执行（RCE）严重漏洞**
- 企业级修复重点：Windows Server、Azure、Office 365
- Cisco SD-WAN CVSS 10.0 完美评分漏洞同期曝光

### Wireshark 多个 RCE 漏洞

Wireshark 被发现存在多个可通过恶意数据包或捕获文件触发远程代码执行的漏洞，安全团队应立即更新至最新版本。
