## 五、💻 编程语言 & DevTools

### 1. Netflix 技术博客：用 JDK Vector API 优化推荐系统

> **来源**：https://netflixtechblog.com/optimizing-recommendation-systems-with-jdks-vector-api-30d2830401ec

**场景**：Netflix Ranker 服务中的"视频新鲜度评分"单项消耗 7.5% CPU。

**优化四步曲：**
1. **从嵌套循环到矩阵乘法**：O(M×N) 独立点积 → 单次矩阵运算
2. **发现反效果**：首次实现反而慢了 5%——`double[][]` 非连续内存 + GC 压力
3. **平坦化缓冲区**：`double[][]` → `double[]` 行主序，消除指针追逐
4. **ThreadLocal 复用**：`ThreadLocal<BufferHolder>` 避免每次请求分配内存

> **Java 后端开发者必读**：这是 JDK Vector API 在生产环境真实落地的典范案例。核心教训——**算法改进如果跟内存布局对着干，就是负优化**。
