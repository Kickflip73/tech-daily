### 1. Google / Addy Osmani 双发布 Agent Skills 标准

GitHub 今日 Trending 被 **Agent Skills** 概念霸榜：

| 项目 | Stars | 核心定位 |
|---|---|---|
| [addyosmani/agent-skills](https://github.com/addyosmani/agent-skills) | 51,732 ⭐ | 生产级工程技能模板（Code Review / Refactor / Test / Security Audit） |
| [google/skills](https://github.com/google/skills) | 13,278 ⭐ | Google 官方 Agent Skill，覆盖 GCP / Android / Firebase |

**深度解读**：这标志着 AI 编码 Agent 进入"技能标准化"阶段。Google 与业界领袖同时推出 Skills 定义，意味着 Agent 的能力不再依赖单次 Prompt 的运气，而是通过可复用、可组合、可验证的 Skill 模块来构建。对 Stark 先生的后端团队而言，这意味着可以：
1. 建立内部代码审查 Skill 库，标准化 CR 质量
2. 将业务逻辑封装为专属 Skill， junior 开发者也能调用专家级能力
3. Skill 的版本管理与 A/B 测试将成为新的工程实践

### 2. EEVEE: 测试时提示学习框架 (HuggingFace 🔥)

- **论文**: [When to Align, When to Predict](https://huggingface.co/papers/2606.11190) + [EEVEE](https://huggingface.co/papers/2606.11182)
- **机构**: Princeton AI² Lab
- **亮点**: 首个多数据集测试时提示学习框架，通过 Router 将输入动态分区到任务集群
- **效果**: 相比 Qwen3-4B 提升 10.38 分，相比 DeepSeek-V3.2 提升 24.32 分，超越 SOTA 方法 GEPA/ACE 达 37-48%

**工程启示**: 提示工程正在从"静态模板"进化为"动态路由 + 测试时优化"，对需要处理异构任务流的微服务体系有借鉴意义。

### 3. 多模态学习相位图：何时对齐、何时预测

- **论文**: [When to Align, When to Predict: A Phase Diagram for Multimodal Learning](https://huggingface.co/papers/2606.11190)
- **核心贡献**: 首次将多模态学习统一为四象限相位图（Both / CA Only / CP Only / Neither），指导从业者在大规模训练前诊断数据集特性
- **验证**: 在立体视觉、图像-文本对、真实天体物理数据上验证

### 4. 多个 Agent 评测基准密集发布

| 基准 | 机构/作者 | 核心关注点 |
|---|---|---|
| ABC-Bench | 多机构 | 生物安全 Agent 能力（液体处理机器人编程、DNA 片段设计、合成筛查规避） |
| CIAware-Bench | Yoshua Bengio 团队等 | 控制干预感知——Agent 能否检测自身轨迹被修改 |
| Workflow-GYM | 多机构 | 长时域 GUI 任务（专业软件操作），最强模型成功率仅 30% |
| TRACE | 清华团队 | Agentic RL 的 Rollout 预算树状分配 |

**趋势判断**: Agent 评测正在从"单点能力测试"转向"安全边界 + 长时域一致性 + 真实专业场景"的三维评估。
