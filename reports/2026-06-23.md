### 5. DeusData/codebase-memory-mcp — 代码知识图谱 MCP Server（今日 ⭐+1,185）

> **仓库**：https://github.com/DeusData/codebase-memory-mcp
> **语言**：C | **协议**：MIT | **⭐总**：10,232+

**这是什么：** 用纯 C 编写的高性能代码智能引擎，通过 tree-sitter AST 分析将代码库索引为持久化知识图谱，为 AI 编码 Agent 提供结构化代码理解能力。

**技术细节：**
- **极端索引速度**：Linux 内核（28M 行代码）3 分钟索引完成
- **158 种语言支持**：vendored tree-sitter grammars 编译进单个静态二进制
- **Hybrid LSP**：9 种语言的语义类型解析
- **120x 更少 token**：5 个结构查询 ~3,400 tokens vs 文件逐个搜索 ~412,000 tokens
- **14 个 MCP 工具**：搜索、追踪、架构分析、影响分析、Cypher 查询、死代码检测等

**为什么值得关注：** AI 编码助手的核心瓶颈是**上下文理解**——如何用最少的 token 让模型理解整个代码库。codebase-memory-mcp 通过结构化知识图谱将 token 消耗降低 120 倍，这是 Agent 基础设施级别的重要突破。

**后端开发者关注点：** 如果你用 Claude Code 或类似工具开发大型项目，这个 MCP Server 可以显著提升代码理解的准确性和效率。纯 C 实现也意味着它足够快、足够轻。

### 🔬 今日学术趋势

1. **Agent 记忆治理**：GateMem 和 MemSlides 都在探索多主体 Agent 的记忆管理问题——当多个 Agent 共享记忆时，如何保证一致性、隔离性和治理？这是 Agent 系统从"单机"走向"集群"的基础问题。
