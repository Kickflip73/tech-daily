### 1. TauricResearch/TradingAgents — 多智能体金融交易框架（今日 ⭐+2,112）

> **仓库**：https://github.com/TauricResearch/TradingAgents
> **论文**：https://arxiv.org/abs/2412.20138
> **语言**：Python | **协议**：MIT

**这是什么：** 一个基于多智能体 LLM 的金融交易框架，模拟真实交易公司的运作模式。通过部署专业化的 LLM 驱动智能体（从基本面分析师、情绪专家、技术分析师到交易员、风险管理团队），协作评估市场状况并做出交易决策。

**核心智能体架构：**
- **基本面分析师**：评估公司财务状况和业绩指标，识别内在价值和潜在风险信号
- **情绪分析师**：分析社交媒体和公众情绪，使用情绪评分算法衡量短期市场情绪
- **新闻分析师**：监控全球新闻和宏观经济指标，解读事件对市场的影响
- **技术分析师**：利用技术指标（如 MACD、RSI）检测交易模式并预测价格走势
- **研究团队**：由看涨和看跌研究员组成，通过结构化辩论平衡潜在收益与固有风险
- **交易员**：基于综合市场洞察决定交易时机和规模
- **风险管理团队**：持续评估投资组合风险，监控市场波动性和流动性
- **投资组合经理**：最终审批/拒绝交易提案

**技术亮点：**
- 基于 LangGraph 构建，支持 LangGraph checkpoint 恢复
- 支持多 LLM 提供商：GPT-5.x、Gemini 3.x、Claude 4.x、Grok 4.x、DeepSeek、Qwen、GLM、Azure OpenAI
- 结构化输出智能体（Research Manager、Trader、Portfolio Manager）
- 持久化决策日志
- Docker 支持 + Windows UTF-8 编码修复

**后端开发者关注点：**
- 展示了如何将复杂业务领域（金融交易）分解为专业化智能体协作
- LangGraph 状态机设计模式值得参考
- 多 LLM 提供商抽象层的设计思路

### 2. warpdotdev/warp — Agentic 开发环境（今日 ⭐+3,401）

> **仓库**：https://github.com/warpdotdev/warp
> **官网**：https://www.warp.dev
> **语言**：Rust | **协议**：AGPL v3 + MIT（UI 框架）

**这是什么：** Warp 是一个从终端诞生的 Agentic 开发环境。它内置了编程 Agent，也支持接入外部 CLI Agent（Claude Code、Codex、Gemini CLI 等）。

**核心特性：**
- **内置编码 Agent**：自动生成代码、修复 bug、重构代码
- **Agent 管理**：支持多 Agent 并行工作，可查看 build.warp.dev 上数千个 Oz Agent 的工作状态
- **Warp Drive**：保存和共享常用命令、工作流
- **现代终端**：基于 Rust 构建，GPU 加速渲染
- **开源代码库**：Warp 客户端代码完全开源，欢迎社区贡献

**技术栈亮点：**
- Tokio（异步运行时）
- NuShell（shell 集成）
- Alacritty（终端仿真器）
- Hyper HTTP 库
- FontKit（字体渲染）

**后端开发者关注点：**
- Warp 的 Agentic 工作流设计理念值得关注
- Rust 构建的终端性能表现优异
- 可以研究其 Agent 编排和任务调度机制
