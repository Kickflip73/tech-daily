### 1. SakanaAI/AI-Scientist-v2 — AI 自主科研系统 v2（今日 ⭐+506，总 ⭐3,433）

> **仓库**：https://github.com/SakanaAI/AI-Scientist-v2
> **论文**：https://pub.sakana.ai/ai-scientist-v2/paper
> **语言**：Python | **机构**：Sakana AI（Tokyo）

**这是什么：** Sakana AI 发布的第二代全自动科研系统。亮点是世界首篇**完全由 AI 撰写并通过同行评审**的 Workshop 论文。v2 彻底移除了对人类撰写模板的依赖，可泛化至整个机器学习领域。

**技术架构：**
- **渐进式 Agentic 树搜索（Progressive Agentic Tree Search）**：与 v1 线性流程不同，v2 通过树状探索路径动态发现和评估假设
- **实验管理 Agent**：独立的 Experiment Manager Agent 负责调度子 Agent 执行实验、收集数据
- **端到端流程**：假设生成 → 代码实验 → 数据分析 → 论文撰写，全流程无人工干预
- **安全警告**：代码在沙箱中运行，因为 LLM 生成的代码可能含危险包调用或无限制网络访问

**v1 vs v2 的 Trade-off：**
- v1：依赖人类模板，成功率高，适合目标明确的任务
- v2：开放式探索，成功率更低，但适合"不知道要发现什么"的真正科研场景

**产业意义：** AI 科研自动化从"辅助"跨入"自主"阶段。未来的 ML Paper Factory 将是超大规模 Agent 集群 24/7 跑实验。已在 ICLR 2025 Workshop 验证可行性。

### 1. Hyperagents — 自我修改的元认知 Agent 框架
> **论文**：https://huggingface.co/papers/2603.19461 | 发布：2026-03-19

**核心思想：** Hyperagents 是一种"自我参照"框架，将任务 Agent 和元 Agent 整合进**单一可编辑程序**中。Agent 不仅执行任务，还能元认知地修改自身行为模式（prompt、工具选择策略）。实现跨领域的开放式自我进化。

**技术细节：** 程序即 Agent，Agent 即程序。通过结构化代码表示 Agent 的 planning + execution + self-modification，可被 LLM 以代码编辑的方式修改自身。

- 维度1：模型参数量（Model Scaling）

**意义：** 训练模型进行更深、更频繁的 Agent-环境交互是下一代研究 Agent 的关键路径。

### 5. PixelREPA — 像素空间扩散 Transformer 的表示对齐
> **论文**：https://huggingface.co/papers/2603.14366

**背景：** REPA（表示对齐）已被证明能加速 Latent Space DiT 训练。但直接用于像素空间（JiT）反而使 FID 变差——因为高维像素空间与压缩语义空间之间存在"信息不对称"。

**解决方案 PixelREPA：**
- 对齐目标变换（Alignment Target Transformation）
- Masked Transformer Adapter（浅层 Adapter + 部分 Token Masking）

**结果：** JiT-B/16 FID 从 3.66 → 3.17，IS 275.1 → 284.6，收敛速度 2x 提升。最终 PixelREPA-H/16 达到 FID=1.81、IS=317.2 的 SOTA 水平。
