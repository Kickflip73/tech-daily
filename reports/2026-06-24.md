### 1. calesthio/OpenMontage — 开源 Agentic 视频生产系统（今日 ⭐+3,592）

> **仓库**：https://github.com/calesthio/OpenMontage
> **总 ⭐**：15,588 | **语言**：Python | **协议**：AGPL-3.0

**这是什么：** 世界第一个开源的 Agentic 视频生产系统，将你的 AI 编程助手（Claude Code、Cursor、Copilot 等）变成一座完整的视频制作工作室。输入自然语言描述，Agent 自动完成**调研 → 脚本撰写 → 分镜设计 → 素材生成 → 剪辑 → 合成 → 自审 → 交付**的完整流程。

**技术架构：**
- **12 条 Pipeline**：覆盖不同视频类型（教程、宣传片、短纪录片、产品展示等），每条 Pipeline 都有预定义的工作流图
- **52 个 Tool**：通过统一工具注册接口，Agent 可自动调用字幕生成、语音合成（Qwen3-TTS、F5-TTS）、图像生成（FLUX、SD）、视频合成（FFmpeg）等工具链
- **500+ Agent Skills**：技能按功能分类，可组合形成复杂的多步骤视频制作流
- **免费/开源工作流**：不依赖付费 API，所有合成、渲染均支持本地或开源模型
- **Agent 协作模式**：支持多个专业子 Agent（导演 Agent、配音 Agent、剪辑 Agent）协同工作

**为什么值得关注：** OpenMontage 证明了"Agent + 工具链"模式在创意工作流的成熟度——不是"AI 辅助写脚本"，而是 Agent 全程主理视频制作。上线数周已超 15k stars，今日仍是 Trending 第一。

**后端开发者关注点：** 如果你在设计 Agentic 应用，OpenMontage 的三层架构（Pipeline / Tool / Skill）是极好的参考。它解决了"Agent 如何编排复杂多步骤工作流"的工程问题，比很多 Agent 框架更贴近生产级设计。

### 2. ZhuLinsen/daily_stock_analysis — LLM 驱动的股票智能分析系统（今日 ⭐+1,119）

> **仓库**：https://github.com/ZhuLinsen/daily_stock_analysis
> **总 ⭐**：47,008 | **语言**：Python | **协议**：MIT

**这是什么：** 一套面向个人投资者的 LLM 驱动多市场股票分析系统，集行情抓取、AI 分析、决策看板和消息推送于一体，核心定位是**零成本、定时运行的智能投资助手**。

**核心能力：**
- **多数据源接入**：同时追踪 A 股（沪深）、港股（00700 等）、美股（AAPL 等）多市场
- **实时新闻聚合**：自动爬取财经新闻，结合 LLM 分析新闻情绪对股价的影响

### 3. garrytan/gstack — YC 总裁的 Claude Code 工程团队 Skills（今日 ⭐+1,011）

> **仓库**：https://github.com/garrytan/gstack
> **总 ⭐**：114,039 | **语言**：TypeScript | **协议**：MIT

**23 个角色包括：**

### 6. DeusData/codebase-memory-mcp — 高性能代码库知识图谱 MCP（今日 ⭐+1,300）

> **仓库**：https://github.com/DeusData/codebase-memory-mcp
> **语言**：C/Go | **协议**：MIT

**这是什么：** 高性能代码智能 MCP Server，将整个代码库索引为持久化知识图谱，毫秒级响应 AI Agent 的代码查询请求。核心解决的问题：AI 编程助手每次都要重新扫描文件树，不仅慢而且消耗大量 Token。

**技术特点：**
- **持久化知识图谱**：用 C 实现核心索引引擎，将代码库中的函数、类、模块、依赖关系全部图式化存储
- **毫秒级索引**：平均仓库在毫秒内完成索引（对比 AI 扫描单文件通常需要数秒）
- **MCP 标准接口**：标准化 MCP Server，接入 Claude Code / Cursor / Copilot 等所有主流工具
- **增量更新**：代码变更时只更新变更部分，无需全量重建

**后端开发者关注点：** 如果你的项目代码量大（10万行+），这个工具能显著提升 AI 编程助手的响应速度和 Token 效率。本质是为 AI 编程工具链补了一个"代码数据库"中间件层。
