## 三、📄 学术前沿：arXiv & HuggingFace Papers

### 🔥 亮点论文 1：Agents-A1 — 35B 参数匹配万亿模型（arXiv: 2606.30616）
> **标题**：Scaling the Horizon, Not the Parameters: Reaching Trillion-Parameter Performance with a 35B Agent

**核心突破**：InternScience 团队证明——**扩展 Agent 的行动时域（horizon），而非模型参数**，可以让 35B MoE 模型达到万亿参数模型（Kimi-K2.6、DeepSeek-V4-pro）的性能。

**技术路径：**
- **长时域知识-行动基础设施**：生成平均 45K token 的轨迹
- **三阶段训练**：全领域 SFT → 领域教师模型 → 多教师域路由策略蒸馏
- **Benchmark 结果**：SEAL-0 (56.4)、IFBench (80.6)、FrontierScience-Olympiad (79.0)

**启示**：Agent 的能力不只取决于基座模型大小，更取决于它能"想多远、做多少步"。这是"小模型 + 长 Horizon"路线的有力证据。

---

### 🔥 亮点论文 2：异步 Pipeline 并行训练的稳定性突破（arXiv: 2606.30634）
> **标题**：One-Step Gradient Delay is Not a Barrier for Large-Scale Asynchronous Pipeline Parallel LLM Pretraining

**核心发现**：AdamW 在异步 Pipeline 并行下严重退化，但 **Muon 优化器展现出强鲁棒性**。引入 Error Feedback 校正后，10B 模型验证了异步与同步训练的性能差距可被弥合。

**后端开发者关注点**：如果你在做大规模 LLM 训练基础设施，Muon 优化器值得关注。它可能是解锁异步 Pipeline 并行训练的关键。

---

### 🔥 亮点论文 3：多轮 LLM 对话中的吸引子态（arXiv: 2606.30571）
> **标题**：Attractor States Emerge in Multi-Turn LLM Conversations

**核心发现**：多轮 LLM 对话中会自然涌现"吸引子态"——对话逐渐收敛到某些固定模式。这对 Agent 长对话设计有重要影响：Agent 的行为可能在多轮交互后"固化"，需要设计扰动机制来保持灵活性。
