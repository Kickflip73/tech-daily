### 2. Bifrost: 混合 TEE-FHE 架构保护 LLM 推理隐私

> **论文**：https://arxiv.org/abs/2606.17421v1
> **作者**：Chenghao Chen 等

**创新点：** 结合可信执行环境（TEE）和全同态加密（FHE），在云上 LLM 推理时保护用户数据隐私。

**技术方案：**
- 线性层用 FHE 在 GPU 加速器上密文执行
- 非线性层、注意力控制逻辑、KV 状态转换在 CPU TEE 中执行
- Bifrost+ 进一步采用 prefill/decode 分离：prompt 侧 KV 状态在 CPU TEE 中构建

**性能：** 在 GPT-2 (1.5B) 上延迟降低 9.25x，LLaMA 3 (8B) 上降低 9.91x。

**后端关注点：** 如果你做 LLM 推理服务且关心数据隐私（金融、医疗场景），这篇论文提供了一个可落地的架构方向。

### 4. ShellGames: LLM 驱动的 SSH 蜜罐欺骗系统

> **论文**：https://arxiv.org/abs/2606.17986v1
> **作者**：Umberto Salviati 等

**创新点：** 用 LLM 驱动 SSH shell 模拟器，与攻击者进行长时间可信交互。结合五项技术：CoT+few-shot 学习、记忆管理、推测执行、沙箱路由、颠覆检测。

**成果：** 命令准确率 0.898（+5.3pp），状态追踪准确率 0.98（+18.3pp），鲁棒性 0.95（+37pp）。20 人用户研究确认其真实感接近真实 shell。

**安全关注点：** 蜜罐是防御重要手段，但传统蜜罐很容易被识别。LLM 驱动的高交互蜜罐可能改变攻防博弈格局。

### 5. PARSE: 防御 LLM Prompt Injection 的新方法

> **论文**：https://arxiv.org/abs/2606.17467v1
> **作者**：Aaditya Pai

**创新点：** 针对企业真实文档的 Prompt Injection 防御。作者发现合成基准上的防御方法在真实文档上失效（Paraphrasing 在真实文档上 p=0.500 无显著效果）。

**PARSE 方案：** 溯源感知检索净化——按注入概率分类每个句子，在重写前提取结构化事实，通过一致性检查循环验证事实保持。

**成果：** 攻击成功率从 25.4% 降至 15.6%（降低 38%），utility 保持 86.9%，且是唯一同时统计显著且保持近基线效用的方案。
